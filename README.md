# Pokemon ROM Patcher (Black 2 / White 2)

A simple but powerful tool for customizing the Pokémon gameplay experience.

## Description

A ROM patching tool for Pokémon Black 2 and White 2 that allows direct modification of in-game data through a user-friendly GUI.

## Features

* Disable EV gain (0 EV mode)
* Configure custom shiny encounter rate
* Enable previously impossible evolutions
* Force Challenge Mode

## Technologies

* Python
* Binary file manipulation
* GUI (Tkinter / custom interface)

## Development Approach

The project was developed with the assistance of AI tools, which were used for:

* debugging and troubleshooting
* improving code structure
* accelerating development workflow

## Technical Challenges

* editing and maintaining ROM data integrity
* working with binary offsets and game data structures
* ensuring stability after applying patches

## Disclaimer

This project does not include any game files. Users must provide their own legally obtained ROM.
